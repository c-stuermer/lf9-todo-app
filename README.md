# Todo Client & Server (v0.1.0)

A minimal, Todo list project consisting of a Vanilla JavaScript frontend and a Python/Flask backend. Developed as a academic project focusing on REST API design.

## Project Status (v0.1.0)
Currently, only the backend REST API (`/server`) is implemented and pushed to this repository.

## Project Structure

The project is divided into two main logical components:
- `/client`: Contains the user interface (HTML, CSS, Vanilla JS).
- `/server`: Contains the REST API and server logic (Python, Flask), including the API specification (`openapi.yaml`).
